##  About The Project
A modern Job Portal web application built using React and Vite.  
Users can browse internship listings, view detailed information, and navigate through multiple pages using dynamic routing.

This project demonstrates clean UI design, reusable components, and proper project structure.

---

##  Features
-  Internship Listings
-  Detailed Internship View Page
-  Login & Signup UI
-  Responsive Layout
-  Fast Performance with Vite
